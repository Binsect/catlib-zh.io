字符串辅助函数(Str)

字符串方法库允许您通过简单的方式访问和操作字符串。

可用方法

  • Method
  • Is
  • AsteriskWildcard
  • Split
  • Repeat
  • Shuffle
  • SubstringCount
  • Reverse
  • Pad
  • After
  • Contains
  • Replace
  • ReplaceFirst
  • ReplaceLast
  • Random
  • Trancate

Method

获取字符串所表达的合法的函数名。

Is

判断规定字符串是否符合匹配表达式。（匹配表达式允许使用星号(*)通配）

AsteriskWildcard

将规定字符串翻译为星号匹配表达式。（即删减正则表达式中除了星号外的所有功能）

Split

根据长度将字符串分割到数组中。

Repeat

将字符串重复指定的次数。

Shuffle

随机打乱字符串中的所有字符

SubstringCount

计算子串在字符串中出现的次数（该函数不计数重叠的子串）

Reverse

反转规定字符串

Pad

把规定字符串填充为新的长度。

  • 如果填充长度小于字符串的原始长度，则不进行任何操作。
  • 填充字符串的两侧。如果不是偶数，则右侧获得额外的填充。

After

在规定字符串中查找在规定搜索值，并在规定搜索值之后返回规定字符串的剩余部分。(如果没有找到则返回规定字符串本身)

Contains

判断规定字符串是否包含规定子字符串。

Replace

在规定字符串中替换匹配项。

ReplaceFirst

替换规定字符串中第一次遇到的匹配项

ReplaceLast

替换规定字符串中从后往前第一次遇到的匹配项

Random

生成一个随机字母（含大小写），数字的字符串。

Trancate

如果长度超过给定的最大字符串长度，则截断字符串。 截断的字符串的最后一个字符将替换为默认为...的省略字符串。
