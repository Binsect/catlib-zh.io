环形缓冲区

环型缓冲区是一种用于表示一个固定尺寸、头尾相连的缓冲区的数据结构，适合缓存数据流。

快速列表实现接口:IRingBuffer

构造环型缓冲区

函数原型
参数 描述
capacity 环状缓冲区的最大容量，为2的次方。如：传入12，则实际生效为16
exposable 是否可以访问内部数组

Capacity

缓冲区的实际最大容量。

函数原型

SyncRoot

同步锁定

函数原型

WriteableCapacity

可以被写入的容量。

函数原型

ReadableCapacity

可以被读取的容量。

函数原型

CanRead

测试是否可以读取指定长度的数据

函数原型
参数 描述
count 希望读取的长度

CanWrite

测试是否可以写入指定长度的数据

函数原型
参数 描述
count 希望写入的长度

GetBuffer

获取原始缓冲区，如果exposable为false则会引发一个UnauthorizedAccessException异常。

函数原型

Read

读取环状缓冲区的数据。

函数原型
参数 描述
buffer 读取到的缓冲区
offset buffer缓冲区的起始偏移量
count 希望读取的长度

Peek

读取环状缓冲区的数据，但是不前移读取位置。

函数原型
参数 描述

offset buffer缓冲区的起始偏移量
count 希望读取的长度

Write

将数据写入到环型缓冲区，返回值为实际写入的长度。

函数原型
参数 描述
buffer 希望写入到环型缓冲区的数据
offset buffer缓冲区的起始偏移量
count 写入的长度
